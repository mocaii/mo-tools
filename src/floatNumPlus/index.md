## API

```typescript
floatNumPlus(a, b);
```

## Params

| 参数 | 说明     | 类型     | 是否必填 | 默认值 |
| ---- | -------- | -------- | -------- | ------ |
| a    | 第一个数 | `number` | 是       | -      |
| b    | 第二个数 | `number` | 是       | -      |

## Result

| 参数  | 说明 | 类型     |
| ----- | ---- | -------- |
| value | 结果 | `number` |

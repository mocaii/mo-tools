# checkIsIntersect

判断多个区间的数字是否相交

## 基本使用

<code src="./demo.tsx" ></code>

## API

```typescript
checkIsIntersect([
  [number, number],
  [number, number],
  [number, number],
]);
```

## Params

| 参数 | 说明     | 类型     | 是否必填 | 默认值 |
| ---- | -------- | -------- | -------- | ------ |
| a    | 第一个数 | `number` | 是       | -      |
| b    | 第二个数 | `number` | 是       | -      |

## Result

| 参数  | 说明 | 类型      |
| ----- | ---- | --------- |
| value | 结果 | `boolean` |

## API

<!-- ```ts

## Params

| 参数 | 说明     | 类型  | 是否必填 | 默认值 |
| ---- | -------- | ----- | -------- | ------ |
| obj  | 参数名称 | `any` | 是       | -      |

## Result

| 参数  | 说明 | 类型      |
| ----- | ---- | --------- |
| value | 结果 | `boolean` |
